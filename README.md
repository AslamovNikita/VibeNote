 VibeNote - Доска для заметок
 Возможности

**Создавай и кастомизируй** заметки с выбором цвета  
**Перетаскивай** карточки по холсту как стикеры на доске  
**Устанавливай дедлайны** – умный автоформат даты (`ДД.ММ.ГГГГ ЧЧ:ММ`)  
**Автосохранение** позиций и содержимого в JSON  
**Оставшееся время** до дедлайна отображается на каждой карточке  
**Контекстное меню** (ПКМ) для быстрого редактирования и удаления
